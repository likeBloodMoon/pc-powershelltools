# pc-cleanuptool
A PowerShell GUI tool for cleaning, optimizing, and configuring Windows with one click. Includes system cleanup, network tools, Windows tweaks, and more.


- System Cleanup

Clean temporary files

Empty Recycle Bin

Clear Windows Update cache

Clear Prefetch cache

Create automatic restore points

- System Repair Tools

Run DISM (/restorehealth)

Run SFC (/scannow)

Run CHKDSK (online scan)

- Network Utilities

Flush DNS cache

Full network stack reset

Set static IP, DNS, subnet mask, gateway

Revert adapter to DHCP

Auto-detect active network adapters

- Windows Preferences

Enable dark mode

Disable Bing search

Show hidden files & file extensions

Disable mouse acceleration

Enable NumLock on startup

- Optional Software Installer

Installs common apps via winget:

Chrome

7-Zip

VLC

VS Code
